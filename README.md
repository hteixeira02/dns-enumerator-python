# DNS Enumeration Tool (Python)

Python tool for DNS enumeration and record discovery using the dnspython library.

## Features
- Enumerates DNS records: A, AAAA, MX, NS, TXT, SOA
- Handles common DNS errors
- Simple and clean implementation

## Requirements
- Python 3.x
- dnspython

## Installation
```bash
pip install dnspython

````

## Usage

python dns_check.py
